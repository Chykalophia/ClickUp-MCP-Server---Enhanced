# Developer Documentation

This directory contains comprehensive guides for developers working with the ClickUp MCP Server Suite.

## Documentation Structure

### Core Guides
- **[Architecture Guide](architecture.md)** - System design, patterns, and technical architecture
- **[Tool Development Guide](tool-development.md)** - Creating and extending MCP tools
- **[Testing Guide](testing.md)** - Testing strategies and best practices
- **[Contributing Guide](contributing.md)** - Contribution workflow and standards

### Quick Start
1. Read the [Architecture Guide](architecture.md) to understand the system design
2. Follow the [Tool Development Guide](tool-development.md) to create new tools
3. Use the [Testing Guide](testing.md) to validate your implementations
4. Review the [Contributing Guide](contributing.md) before submitting changes

### Code Examples
See the `/examples/` directory for working implementations and integration patterns.

## Getting Help

- Check existing documentation first
- Review code examples in `/examples/`
- Open an issue for bugs or feature requests
- Follow the contributing guidelines for pull requests
